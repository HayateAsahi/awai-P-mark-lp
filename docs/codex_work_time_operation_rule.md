# Codex 作業開始・終了時の記録ルール

## 目的
Codex が作業を開始した時点と終了した時点を、`docs/codex_work_time_log.md` に記録するための運用ルールです。

## Codex への指示内容
以下のルールに従って記録してください。

### 1. 作業開始時
- ユーザーから「作業開始」の連絡があったら、`docs/codex_work_time_log.md` に新しい行を追加する
- この時点では以下を記録する
  - 日付
  - 作業内容
  - 開始時間
- 終了時間、合計作業時間は空欄のままにする

### 2. 作業終了時
- ユーザーから「作業終了」の連絡があったら、開始時に追加した同一作業の行を更新する
- この時点で以下を記録する
  - 終了時間
  - 合計作業時間
  - 必要なら備考

### 3. 時刻と計算ルール
- 日付は `YYYY-MM-DD` 形式
- 時刻は `HH:MM` 形式
- 合計作業時間は、開始時間から終了時間を差し引いて算出する
- 記録は日本時間を基準とする

### 4. 記録フォーマット
`docs/codex_work_time_log.md` の表形式に従うこと

### 5. 運用上の注意
- 既存の行を勝手に上書きしない
- 対応中の作業が複数ある場合は、作業内容で識別する
- 開始連絡がない作業は、勝手に新規記録しない
- 終了時は必ず開始済みの行に追記する

## Codex にそのまま渡す用の指示文

```md
今後、私が「作業開始」と伝えたら、docs/codex_work_time_log.md にその作業の開始日時を記録してください。

私が「作業終了」と伝えたら、対応する行に終了日時と合計作業時間を追記してください。

記録ルールは docs/codex_work_time_operation_rule.md に従ってください。
日付は YYYY-MM-DD、時刻は HH:MM、日本時間基準で記録してください。
既存ログは消さず、追記・更新で管理してください。
```
